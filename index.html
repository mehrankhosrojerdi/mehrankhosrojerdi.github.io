Machine Learning Projects

Binary Classification with a Bank Churn Dataset

The dataset for this project was generated from a deep learning model trained on the Bank Customer Churn Prediction dataset. The goal of this project is to predict whether a customer continues with their account or closes it.

Ref: Walter Reade, Ashley Chow. Binary Classification with a Bank Churn Dataset. Kaggle, 2024. https://kaggle.com/competitions/playground-series-s4e1

Multi-Class Prediction of Obesity Risk

The dataset for this competition was generated from a deep learning model trained on the Obesity or CVD risk dataset. The goal of this project is to use various factors to predict obesity risk in individuals, which is related to cardiovascular disease.

Ref: Walter Reade, Ashley Chow. Multi-Class Prediction of Obesity Risk. Kaggle, 2024. https://kaggle.com/competitions/playground-series-s4e2

Steel Plate Defect Prediction

The dataset for this project (both train and test) was generated from a deep learning model trained on the Steel Plates Faults dataset from UCI. and the goal of this project is to Predict the probability of various defects on steel plates.

Ref: Walter Reade, Ashley Chow. Steel Plate Defect Prediction. Kaggle, 2024. https://kaggle.com/competitions/playground-series-s4e3

AI Tools for Quantum Machine Learning

PyTorch is an open-source machine learning framework developed by Facebook's AI Research lab. It provides dynamic computational graphs and tensors, making it popular among researchers and developers. Check out the PyTorch GitHub repository for more information. You can also start with a basic handout I created as a repository .

TensorFlow is an open-source ML framework developed by the Google Brain team. It is designed to facilitate the development and training of ML models, particularly deep learning models. TensorFlow provides a comprehensive, flexible ecosystem of tools, libraries, and community resources that allows researchers and developers to build and deploy ML applications effectively. Check out the TensorFlow GitHub repository for more information.
